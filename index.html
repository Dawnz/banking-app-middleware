Welcome to the Bank App API

API Endpoints

The endpoints of the API starts at the base route:

  • /accounts

    • Has full CRUD functionality for managing the accounts

  • /users

    • Has full CRUD functionality for managing the users. Users would created at this endpoint as well/ registered.

  • /transactions

    • Full CRUD functionality for managing transactions done to an account.

  • /authenticate

    • Logs in the user and is incharge of authorizing access to the API.

API Requirements:

This API was created with the following requirements in mind

  • Allow the user to create a new account
  • Allow Deposits to be made to any account as long as an account is provided.
  • Allow withdrawals from an account but only the owner of the account can perform this action.
  • Show the balance and calculate balance based on the transactions done on the account
  • Allow user to login with their credentials, Username and Password
  • Allow user to create an online profile for their accounts
  • List all accounts tied to a user
